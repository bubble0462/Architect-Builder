# Changelog

All notable changes to Architect Builder will be documented in this file.

## 1.0.0 - 2026-06-08

### Added

- Added `plantask` skill for generating planning handoff files.
- Added `executetask` skill for executing planned tasks.
- Added `reviewtask` skill for reviewing implementation results.
- Added Claude Code plugin marketplace support through `.claude-plugin/marketplace.json`.
- Added Claude Code plugin manifest through `.claude-plugin/plugin.json`.
- Added bilingual documentation in English and Simplified Chinese.
